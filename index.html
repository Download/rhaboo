The easiest, fastest and most portable persistence solution on the web

Easy

Only three functions in the API, of which you'll need two.

Complete

Stores objects nested to any level. Supports non-numeric properties of arrays.

Fast

Stores components of objects separately rather than just stringifying and parsing the whole thing.

Portable

Only relies on localStorage.

For instance: 

    <script src='rhaboo.min.js'></script>
    <script>
      var store = new Rhaboo.Persistent('Some name');
      store.write('counter', store.counter ? store.counter+1 : 1);
    </script>
    <body onLoad='document.getElementById("count").innerHTML = store.counter;'>
      <p>You saw this <span id='count'></span> times</p>
    </body>

That is quite literally all you need to know, but you can also try: 

    store.write('somethingfancy', {
      one: ['man', 'went'],
      2: 'mow',
      went: [  2, { mow: ['a', 'meadow' ] }, {}  ]
    });
